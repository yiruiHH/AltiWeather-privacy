1. 适用范围

本隐私政策适用于云途应用。云途用于路线规划时查看沿途天气、海拔和穿衣建议。

2. 我们收集和使用的信息

云途不提供账号注册功能，不主动收集姓名、手机号、身份证号、通讯录、照片、音频、视频或支付信息。

为提供核心功能，应用可能在本地或通过第三方服务处理以下信息：

  • 位置信息：在您授权定位权限后，用于获取当前位置、设置路线起点和展示地图位置。
  • 地点搜索和路线信息：您输入的起点、终点、搜索关键词和路线坐标用于地点搜索、驾车路线规划和地图展示。
  • 天气和海拔查询信息：路线采样点的经纬度会用于查询沿途天气、体感温度、降水、风速和海拔。
  • 本地设置：应用会在设备本地保存天气采样间隔等偏好设置。
  • 设备和网络状态：地图和定位 SDK 可能为定位、地图加载、网络连接判断及服务稳定性处理必要的设备和网络信息。

3. 权限说明

  • 定位权限：用于获取当前位置和提供路线规划功能。您可以拒绝或在系统设置中关闭该权限，关闭后与当前位置相关的功能可能不可用。
  • 网络权限：用于加载地图、搜索地点、规划路线，以及查询天气和海拔数据。
  • 存储相关权限：Android 旧版本系统可能由地图 SDK 声明相关权限，用于 SDK 正常运行；应用本身不读取您的个人文件。

4. 第三方服务

为实现地图、定位、地点搜索、路线规划、天气和海拔功能，云途使用以下第三方服务。第三方服务可能按照其隐私政策处理必要数据。

  • 高德开放平台 / AMap SDK：用于 Android 版地图展示、定位、地点搜索和路线规划。隐私政策：https://lbs.amap.com/pages/privacy/
  • Open-Meteo：用于天气和海拔查询。隐私政策：https://open-meteo.com/en/privacy
  • Apple MapKit 和 CoreLocation：用于 iOS 版地图、地点搜索、路线规划和定位。Apple 隐私政策：https://www.apple.com/legal/privacy/

5. 信息存储和共享

云途没有自建服务器，不运营账号系统，也不会出售您的个人信息。应用处理的数据主要用于在您的设备上展示路线、地图、天气、海拔和穿衣建议。为实现这些功能，必要数据可能会发送给上文列明的第三方服务。

6. 信息安全

我们会尽力使用合理措施保护信息安全。请理解，互联网传输和移动设备运行环境无法保证绝对安全。

7. 您的隐私权利

您可以通过以下方式管理您的信息和权利：

  • 在系统设置中开启或关闭云途的定位权限。
  • 清除应用数据或卸载应用，以删除保存在设备本地的偏好设置。
  • 停止输入地点搜索关键词或停止使用路线规划功能，以避免继续产生相关查询。
  • 如需咨询、投诉或行使个人信息相关权利，请通过 GitHub 仓库联系开发者：https://github.com/yiruiHH/AltiWeather/issues

8. 未成年人保护

云途面向一般用户提供路线天气查询功能。若您是未成年人，请在监护人同意和指导下使用本应用。

9. 政策更新

我们可能根据功能变化、法律法规或审核要求更新本隐私政策。更新后的政策将在本页面发布，并以页面显示的更新日期为准。

English summary

云途 uses location, place search, route coordinates, weather/elevation query coordinates, network access, and local preferences only to provide route weather, elevation, map display, routing, and clothing recommendations. The app does not provide accounts, does not operate a custom backend, and does not sell personal information. Third-party services include AMap, Open-Meteo, and Apple MapKit/CoreLocation where applicable.
